## Getting Started

To get started:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sstricklen112/capstone_example.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd capstone_example
   ```
3. **Open the notebooks** using Jupyter:
   ```bash
   jupyter notebook
   ```
